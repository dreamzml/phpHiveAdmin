phpHiveAdmin
============

This is a HIVE(apache) admin and query tool written in php

It can simply use like phpMyAdmin

Program development environment:

CentOS 5.4
Hadoop-0.20.203.0
Hive 0.7.1
php 5.3.8
apache 2.2.x
